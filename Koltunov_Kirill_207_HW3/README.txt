Эта программа реализует игру в 21 с шулерами и обычными игроками по заданным параметрам.

Реализовано все на 10. Ввод осуществляется через консоль после запроса от программы.
Изначально выводил ход игры в консоль на каждом шаге, но после уточнений от Ефима Михайловича
добавил флаг в классе Game, поднимая или опуская который соответственно можно задать вывод или его отсутсвие.

Добавил два jar файла. В main.jar не осущетсвляется вывод информации о состоянии игры.
Для запуска пропишите: java -jar main.jar

В mainWithOutput.jar осущетсвляется вывод информации о состоянии игры в консоль.
Для запуска пропишите: java -jar mainWithOutput.jar

Весь код задукоментирован, поэтому вопросов не должно возникнуть!
